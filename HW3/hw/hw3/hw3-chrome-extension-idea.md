## 1. Extension Concept
This Chrome extension idea simplifies online shopping by allowing its users to automate the process of checkouts with a single click of a button. By clicking it, the user's shipping and billing information, preferred delivery options, and secured payment details are automatically filled with in the website. This extension saves users time and effort during online purchases and significantlyu helps on repeated purchases.

## 2. Key Features
1. One-Click Checkout Button:   
  - Automatically inserts stored shipping and billing information into checkout forms.  
  - By clicking the button, the user places the order instantly.  
  
2. Secure Auto-Fill Storage:  
  - Saves user information locally in a secured format to ensure privacy.  
 
3. Quick Toggle: 
  - Icon in the extension that allows users to enable or disable autofill functionality at any time.

## 3. Research
1. Existing Extension:   
  - Dashlane and LastPass 
  - Currently provide autofill and password management functionality but not soecifically for easyt checkouts.  
  
2. How This Extension Differs:  
  - Focused specifically on speeding up online purchases, not password management like Dashlane or LastPass.  
  - Adds one-click checkout for faster transactions while keeping credit card information safe.  
  - Also, it automatically selects delivery preferences, which existing autofill tools do not do (slected by the user previously and can be changed whenever the user prefers to change it).  
  - Save a lot of time and effort for repeated purchases like ordering household stuff every 2 weeks.
